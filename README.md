# ZADM 随机点名系统

**ZADM**（子昂点名）是一个基于 Python Tkinter 的图形化随机点名工具，界面简洁、操作直观，支持从 JSON 文件动态加载姓名列表，提供**滚动抽选**、**名单管理**、**语法高亮代码查看**等实用功能，适合课堂互动、活动抽奖等场景。

---

## ✨ 主要特性

- 🎲 **随机滚动点名**：姓名快速滚动，点击“停止”即随机抽选并锁定一名成员
- 🌍 **中英文居中显示**：无论中文姓名还是长英文名，均在窗口中央恒稳展示
- 📋 **可视化名单管理**  
  - 批量添加（一行一个姓名，自动去重）  
  - 多选框批量删除（勾选后一键移除）  
  - 列表支持鼠标滚轮滚动，长名单浏览无压力
- 📄 **内嵌代码查看器**  
  - 暗色主题语法高亮（关键字加粗，字符串 / 注释 / 数字分色）  
  - 实时读取并展示当前程序源码
- 💾 **JSON 数据持久化**：姓名列表保存为 `data/names.json`，结构清晰，可手动编辑
- 🖥️ **窗口控制**：主窗口置顶、固定尺寸、可自定义程序图标

---

## 🛠️ 运行环境

- **Python 3.6** 及以上版本
- **纯标准库**：`tkinter` `json` `os` `re` `random`，无需任何第三方依赖

## 🚀 快速开始

### 1. 获取代码

```bash
git clone https://github.com/yourusername/ZADM.git
cd ZADM
```
##2. 准备姓名数据
在 data/ 目录下创建 names.json（若不存在程序会自动生成默认名单），内容为 JSON 数组：
```json
[
  "张三",
  "李四",
  "王五",
  "赵六"
]
```
## 3.运行程序
```bash
python main.py
```
## 姓名管理窗口
-批量添加：在文本框中每行输入一个姓名，点击“添加到名单”（已存在的姓名会自动忽略）

-批量删除：勾选需要删除的姓名前的复选框，点击“删除选中”

-列表内容较多时，可使用鼠标滚轮流畅翻阅

## 代码查看器
-窗口自动显示 当前项目源码（main.py），并带有 VS Code 风格暗色语法高亮

-文本只读，支持滚动查看，方便展示或检查代码

## ⚙️ 自定义配置
修改窗口图标：替换 E:\data\zadm.ico 为你的图标（可修改 main.py 中的路径）

修改姓名文件路径：编辑 main.py 中的 json_path 变量（默认 data/names.json）

调整滚动速度：修改 scroll_name() 中的 after(50, scroll_name) 毫秒值

更改默认颜色/字体：在 main.py 中搜索相关 Label 参数进行调整
## ❓ 常见问题
Q：英文名字显示偏右怎么办？
A：程序已使用相对定位 (relx=0.5, anchor='n')，确保姓名始终水平居中。若仍异常，请检查是否更改了相关代码。

Q：JSON 文件报错怎么办？
A：请检查 names.json 格式，必须为严格的 JSON 数组：姓名用双引号包裹，每个元素后用英文逗号分隔，最后一个后不加逗号。可使用在线 JSON 校验工具排查。

Q：鼠标滚轮在名单管理中无法滚动？
A：请确保 name_manager.py 已绑定 MouseWheel 事件（最新版已修复）。

Q：如何让窗口可调整大小？
A：编辑 main.py，将 w.resizable(False, False) 改为 True。
